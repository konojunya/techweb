##Twitter APIとは

APIとは、twitterに限らずいろいろなwebサービスが、広くOPENに
いろいろなかたがたが、サービスを利用できるように機能を開放している
ことを指します。
twitterで言いうと、

- twitterへの投稿
- twitterの検索
- twitter友達関係表示

などなどです。

これを利用して、個人や企業が楽しい便利なtwitterアプリを
開発しています。
